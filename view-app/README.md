# vue-crud-app

Набросок административной части для тестирования REST API.
Т.к. задача ориентирована на backend, то административная часть сделана крайне упрощённо, но используя прилагаемое API.

## Настройка
Подправить путь и доступы к API в файле .env (по умолчанию предполагается, что API находится по тому же адресу, что и административная часть)

## Установка
```
npm install
```

## Компиляция
```
npm run build
```

## Перенос в YII
1. Из папки `/dist/` скопировать папки `/js/` и `/css/` в корневую папку `/web/` проекта
2. Из папки `/dist/` скопировать файл `index.php` в корневую папку `/views/layouts` проекта, с именем `main.php`
